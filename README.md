# Blue Sky Thinking

# Quick start
Install Docker Compose: https://docs.docker.com/compose/

Start the backend and frontend services by:
```shell
./bin/up.sh

# Or
docker-compose up
```

## Cleaning up

Shutdown the network if the application is misbehaving:

```shell
./bin/down.sh
```
